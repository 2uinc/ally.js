Fix Pointer Focus Children Browser Bug

ally.js provides the method ally/fix-browser/pointer-focus-children to workaround an issue in Internet Explorer 10 and 11 where focusable elements with display: flex; would inherit bing focusable onto their children (not descendants).

This Workaround will only engage for Internet Explorer 10 and 11. No other browser is burdened with this (aside from having loaded the code).

Usage

// load fixPointerFocusChildren()
var fixPointerFocusChildren = require('ally/fix-browser/pointer-focus-children');
// acquire context
var context = document.getElementById('element-to-fix');

// activate the fix on a specific sub-tree (for every browser)
var disengage = fixPointerFocusChildren(context);

// activate the fix for the entire document (for every browser)
var disengage = fixPointerFocusChildren();

// disable the fix
disengage();
