Focus Compatibility Tables

The data displayed in the tables of this document was gathered by manually running the focusable test. I've not employed automation (WebDriver) for this because I wanted the tests to start in the browser's chrome (the browser UI itself) and I didn't see how this could be managed reliably with current test automation tools.
