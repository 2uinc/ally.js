Which Elements Are Focusable, Which Are Tabbable?

The following table details if an element can be focused or even tabbed per browser. It is a static copy of focusable/table.html.

focusable
The element is focusable, not tabbable
tabbable
The element is focusable and tabbable
tabbable
The element is tabbable, not focusable
no
The element is neither focusable nor tabbable
blue border
The element's focusable/tabbable state deviates from the expected state
